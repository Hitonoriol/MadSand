### How to build

**Game**

```
./gradlew texturePacker desktop:dist
```

Output file: `desktop/build/libs/MadSand-<VERSION>.jar`  

\
**Launcher**
```
./gradlew launcher:dist
```

Output file:  `launcher/build/libs/MadSandLauncher-<VERSION>.jar`
